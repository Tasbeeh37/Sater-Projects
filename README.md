## Body Mass Index (BMI) Calculator

The **BMI** (Body Mass Index) is a measurement used to estimate body fat based on a person’s **weight** and **height**.

It helps classify individuals into different weight categories to indicate their level of body fat.

### BMI Formula

BMI is calculated using the following formula:

```
BMI = weight / (height × height)
```

Where:

* **Weight** is measured in kilograms (kg)
* **Height** is measured in meters (m)

---

## BMI Classification

| Category      | BMI Value      |
| ------------- | -------------- |
| Underweight   | Less than 18.5 |
| Normal Weight | 18.5 to 24.9   |
| Overweight    | 25 to 29.9     |
| Obese         | 30 or more     |
